How to checkin code to GITHUB
1. go to master branch (git checkout master)
2. git pull
3. Create / switch to a local branch
4. do the file changes
5. git status(to check the modified file)
6  git add "filename" (for all file use git add -a )
7  git commit -m "comment"
8 git push origin "branch"

go to github(browser)
create pull request
do the code review
merge the code from broswer

git checkout master
